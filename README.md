## Installation

- Download and import the managed solution `WebresourceEditorAPI_managed.zip` from the `solution` folder of this project.
- Then import the managed solution `WebresourceEditor_managed.zip`.
- Open the HTML file (`<your_org_here>/WebResources/web_editor.html`).
- Enter any webresource name you want to edit.
- Press CTRL+S to save and CTRL+P to publish.
